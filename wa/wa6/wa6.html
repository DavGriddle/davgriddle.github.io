CSS Page Design

Color Pallet

Here is the color pallet used in this design:

  • Primary Color: #062a3d (11.98:1)
  • Secondary Color: #064262 (8.58:1)
  • Accent Color: #137166 (4.7:1)
  • Background Color: #e9e6e0

Typography

  1. Font Family: "Zalando Sans Expanded", sans-serif
  2. Heading 1: 3rem, 700 weight, primary color
  3. Heading 2: 2rem, 600 weight, secondary color
  4. Heading 3: 1.5rem, 500 weight, accent color
  5. Body Text: 1rem, default weight, default color

Spacing System

My spacing scale uses consistent multiples of 8 for clean layout rhythm.

--space-1 = 8px
--space-2 = 16px
--space-3 = 24px
--space-4 = 32px
--space-5 = 48px

Layout

The main content area uses a max-width of 1200px and is centered with auto margins. The page has consistent padding and box-sizing set to border-box for easier layout control.
